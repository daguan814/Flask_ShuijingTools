only

非常基础也终极简单

是 one +形容词跟副词后缀 -ly

组成的复合词

one 是幼儿园入学必考词

意思是

一

譬如韩寒主创的文艺装逼小清新app

「ONE·一个」

的名字就是这个

-ly 这个后缀

可以表示

形容词后缀 xx的

跟

副词后缀 xx地

one 后面衔接后缀 -ly 的时候

无意义且不发音的e要自行脱落

所以剩下的事情就很明了了

only这个词有两个意思

一的

跟

一地

再稍微展开一点

就是

唯一的

跟

唯一地

这就是only这个词的意思


然后给大家光速秒杀下系列词


onliest

英 /'əunliist, 'ɔ:n-/  美 /'əunliist, 'ɔ:n-/

only+最高级后缀est

所以是 最唯一的

也衍生出 最合适的 最好的 的意思

你的鱼塘里可以有很多鱼

备胎连起来可以绕地球两圈

但倘若某人之于你是那个唯一的唯一

对你来说

Ta必然就是最合适也是最好的那个



read-only

英 /'ri:dəunli/  美 /'ri:dəunli/

only=只

read=读

所以意思就是 只读

只能读取不能修改的意思

read-only file

就是只读文件的意思
