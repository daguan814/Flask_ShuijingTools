今天给大家撸下iso系列

「iso」这个前缀

最初的形式是「isso」

是短语「is so」的连写

众所周知

单词可以繁琐

词根必须精简

重复跟冗余的「s」既不影响发音也不影响意思

所以中间重复跟冗余的「s」只保留一个就好


「so「是小学一年级必考词

意思是「相同 一样」

随便给大家举几个例子

譬如

「He likes English, so do I.」

「他喜欢英语，我也一样 / 我也相同。」

「He is a Chinese, so am I.」

「他是个中国人，我也一样 / 我也相同。」

「I think so.」

「我觉得也是一样 / 我觉得也相同。」

所以「is so」直译过来就是

「是一样的 / 是相同的」

前缀「iso」只是「is so」的缩略形式

作为前缀依然是「相同 一样」的意思


然后给大家光速秒杀下系列词
