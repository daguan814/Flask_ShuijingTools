今天给大家撸下window系列

统治整个操作系统界的 windows 系统

无人不知

windows 是 微软 研发的 多窗口操作系统

有别于之前一直在沿用的 DOS 提示命令符的操作系统

需要输入繁琐的指令跟代码才可以实现对应的功能

在 windows 多窗口操作系统里

一切本需要输入代码才可以实现的功能都呈现为可重叠的多窗口

大家只需要控制鼠标打开对应的窗口

不需要记忆繁琐的代码跟指令

所以风靡全球

所以 windows 温豆思 作为多窗口操作系统

s 是复数 表示 多

window 就是 窗口 的意思

撸过一遍永生难忘之


然后给大家光速秒杀下系列词


window

英 /ˈwɪndəʊ/  美 /ˈwɪndoʊ/

本身即单词

就是 窗户 的意思



windowed

英 /ˈwɪndəʊd/  美 /'windəud/

window 窗户

-ed 是形容词后缀 表示 有xx的

所以是 有窗的 意思



shopwindow

英 /ˌʃɒpˈwɪndəʊ/  美 /ˈʃɑːpˌwɪndoʊ/

shop 商店 商铺

window 窗

能透过去看到商场摆放的吧商品的窗子 就是 橱窗 陈列创

所以 shopwindow 是 陈列窗 橱窗 的意思



windowsill

英 /ˈwɪndəʊsɪl/  美 /ˈwɪndoʊsɪl/

window 窗 窗口

sill 窗台 基石 门槛

所以是 窗台 窗沿 的意思



windowy

英 /'wɪndəwɪ/  美 /'wɪndəwɪ/

window 窗户

-y 是形容词后缀 表示 多xx的

所以是 窗多的 意思



multiwindow

英 /mʌlti ˈwɪndəʊ/  美 /mʌlti ˈwɪndoʊ/

multi 多xx

window 窗户

所以是 多窗口 的意思



info window

英 /ˈɪnfəʊˈwɪndəʊ/  美 /ˈɪnfoʊˈwɪndoʊ/

info 信息 情报

window 窗 窗口

所以是 信息窗口 的意思



side-window

英 /saɪdˈwɪndəʊ/  美 /saɪdˈwɪndoʊ/

side 侧面的

window 窗户

所以是 侧窗 的意思



window frame

英 /ˈwɪndəʊ freɪm/  美 /ˈwɪndoʊ freɪm/

window 窗 窗口

frame 框架 结构

所以是 窗口框架 的意思
