今天给大家撸下dread系列

「dread」翻译学直译过来发音 =「惴的」[zhuì de]

「惴惴不安」这个成语是中高考语文的必考词

但凡是有接受过义务教育的童鞋就必然都知道

「惴惴不安」就是「因恐惧或担忧而不安」的意思

譬如

「自从被检查出新冠 老王整日惴惴不安」

「惴」[zhuì] 这个字就是「恐惧」&「担忧」的意思

「dread」翻译学直译过来发音 =「惴的」

作为词根的含义也是「惴的」

也就是「恐惧的 担忧的」的意思

音义相同

只要读出来就可以秒懂是什么意思

撸过一遍永生难忘之


然后给大家光速秒杀下系列词


dread

英 /dred/  美 /dred/

本身即单词

就是「令人恐惧的 令人担忧的」的意思

除了形容词之外

还可以做动词跟名词

「v. 恐惧 担忧」

「n. 恐惧 担忧」

「 I dread the future.」

「  我担忧未来。」

「 Lisa, I dread it so terribly.」

「  丽萨，我非常担忧。」



dreadful

英 /ˈdredfl/  美 /ˈdredfl/

「dread」是「恐惧 担忧」的意思

「-ful」是形容词后缀「xx的」

所以是「令人恐惧的 令人担忧的 极其糟糕的」

「 Looks dreadful.」

「  看起来令人恐惧的。」

「 Dreadful man.」

「  令人担忧的男人。」

「 Oh, how dreadful!」

「  哦，多令人恐惧啊！」



dreadfully

英 /ˈdredfəli/  美 /ˈdredfəli/

「dreadful」+「ly」副词后缀「xx地」

所以是「令人恐惧地 令人担忧地 极其糟糕地」

「 This is going dreadfully.」

「  情况正变成极其糟糕地了。」

「 get immediately dreadfully.」

「  突然间变得极其糟糕。」

「 Dreadfully unproductive offices.」

「  令人担忧地无价值的办事处。」



dreadless

英 /'dredlɪs/  美 /'dredlɪs/

「dread」是「恐惧 担忧」的意思

「-less」是形容词后缀「不xx的」

所以是「不恐惧的 不担忧的 不怕的 无畏的」
