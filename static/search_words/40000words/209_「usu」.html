给今天给大家光速秒杀下usu这个词根跟所有系列词

「usu」翻译学直译过来发音 =「无殊」

也就是「没有特殊情况」

「一如寻常 / 一如通常」

也就是「寻常 通常」的意思

只要读出来就可以秒懂是什么意思

撸过一遍永生难忘之


然后给大家光速秒杀下系列词


usual

英 /ˈjuːʒuəl/  美 /ˈjuːʒuəl; ˈjuːʒəl/

「usu」是「寻常 通常」的意思

「-al」是形容词后缀「xx的」

所以是「寻常的 通常的」

「 -The usual? -Yeah.」

「  -还是点和以前一样的吗？-没错。」

「 Same as usual.」

「  和平常一样。」

「 Usual boring pageantry.」

「  寻常且无聊的隆重仪式。」



usually

英 /ˈjuːʒuəli/  美 /ˈjuːʒuəli; ˈjuːʒəli/

「usu」是「寻常 通常」的意思

「-al」是形容词后缀「xx的」

「-ly」是副词后缀「xx地」

多重后缀只看最后一个

所以是「寻常地 通常地」

「 I don't usually drink.」

「  我一般不喝酒。」

「 It's usually temporary.」

「  这通常是暂时的。」

「 I'm usually so graceful.」

「  我通常是很优雅的。」



unusual

英 /ʌnˈjuːʒuəl/  美 /ʌnˈjuːʒuəl; ʌnˈjuːʒəl/

「un」表示否定

「usu」是「寻常 通常」的意思

「-al」是形容词后缀「xx的」

所以是「不寻常的 少有的 罕见的」

「 An unusual disease.」

「  一种罕见的疾病。」

「 You'are an unusual person.」

「  你真是一个不寻常的人。」

「 It's an unusual combination.」

「  这是一种不寻常的结合。」



unusually

英 /ʌnˈjuːʒuəli; ʌnˈjuːʒəli/  美 /ʌnˈjuːʒuəliˌʌnˈjuːʒəli/

「un」表示否定

「usu」是「寻常 通常」的意思

「-al」是形容词后缀「xx的」

「-ly」是副词后缀「xx地」

多重后缀只看最后一个

所以是「不寻常地」

「 Louis was unusually nervous.」

「  路易斯不是一般地紧张。」

「 You do seem unusually lighthearted today, Josh.」

「  乔什，你今天似乎无比寻常地开心呀。」

「 Such unusually bad luck.」

「  不是一般地倒霉。」



as usual

英 /æz ˈjuːʒʊəl/  美 /æz ˈjuːʒʊəl/

「as」是「和xx一样」的意思

「usual」是「寻常的」的意思

所以是「和寻常的一样 像往常一样」

「 Nothing, as usual.」

「  没什么事，和平常一样。」

「 As usual, Grace's right.」

「  和往常一样，格蕾丝说的对。」

「 Somewhat happier than usual or as happy as usual?」

「  是和平常相比有点开心呢，还是和平常一样开心呢？
