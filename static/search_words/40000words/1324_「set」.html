今天给大家光速秒杀下set这个词根跟所有系列词

「set」头部的「se」翻译学直译过来发音 =「设」

众所周知

「设」即「设放」

也就是「放置」的意思


结尾 + 万能 t 不改变意思

「set」作为词根的含义也是「设」

也就是「放置」的意思

音义一致

只要读出来就可以秒懂是什么意思

撸过一遍永生难忘之


然后给大家光速秒杀下系列词


set

英 /set/  美 /set/

本身即单词

就是 放置 的意思

也可以表示 设置 的意思

设置 本意也是 放置 的意思

器物之须 设置如式

就是 器物之须 放置如式

是完全一样的意思

可以完全等义替换之



inset

英 /ˈɪnset/  美 /ˈɪnset/

in 向内 进入

set 放置

放置进去

就是 嵌入 镶入 的意思

作为名词就是 镶入物 嵌入物 的意思



reset

英 /ˌriːˈset/  美 /ˌriːˈset/

re 重新

set 设置

所以是 重新设置



preset

英 /ˌpriːˈset/  美 /ˌpriːˈset/

pre 预先

set 设置

所以是 预先设置



setter

英 /ˈsetə(r)/  美 /ˈsetər/

set 设置

双写 t 无意义

-er 是名词后缀 xx者

所以是 设置者 安排者
