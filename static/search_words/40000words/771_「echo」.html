今天给大家光速秒杀下echo跟所有系列词

「Echo」是古希腊神话中的「回音女神」

无论是人还是神

id 即一切

叫什么那你就得是什么

所以「echo」必然就是「回音」的意思

众所周知

宙斯的夫人赫拉是一位嫉妒心极重的更年期妇女

「echo」小姐姐的美貌让她醋意大发

于是降下天罚

让「echo」从此以后都没办法再正常讲话

无论别人对她说什么

「echo」只能重复这段话的最后三个字

都说人类的本质的是复读机

「echo」小姐姐这下直接变成了复读机中战斗机

「回音」就是最极致的复读

无论你说什么都只会被回以同样的声音

「回音」小姐姐心里苦

但是「回音」小姐姐却说不出

因为「回音」小姐姐只能复读

实在是鬼畜

「echo」翻译学直译过来也 =「异口」

作为小学一年级必考词

「异口同声」的意思就是「不同的嘴说出同样的话」

360 度全方位无死角完美符合「回音女神」的人设

无论别人说什么

「echo」都会以异于别人之口回以相同之声

「异口同声」就是「echo同声」

也就是「echo」回以相同之声

所以「异口同声」&「Echo同声」的「echo」就是「回音」的意思

撸过一遍永生难忘之



然后给大家光速秒杀下系列词


echo

英 /ˈekəʊ/  美 /ˈekoʊ/

本身即单词

就是 回音 的意思

「 Listen to the echo.」

「  听回音。」

「 That's an echo, sir.」

「  那是回音，长官。」

「 There's an echo.」

「  有回音。」



echoic

英 /eˈkəʊɪk/  美 /eˈkoʊɪk/

echo 回音

-ic 是形容词后缀 xx的

所以是 回音的

「 By the way there are such things as echoic memory.」

「  顺便说一下，有回声的记忆这种东西。」

「 Because this memory is called echoic memory.」

「  因为这种记忆叫做回声的记忆。」

「 It's called probably echoic memory.」

「  这可能叫做回声的记忆。」



reecho

英 /riːˈekəʊ/  美 /riˈeko/

re 再

echo 回音

再回音

也就是 再回响 回荡 的意思

「 And I do want to reecho.」

「  我确实想回荡一下。」



anechoic

英 /ˌænɪˈkəʊɪk/  美 /ˌænɪˈkoʊɪk/

an 表示否定 无xx

echoic 回声的

所以是 无回声的

「 False-bottom with anechoic coating to absorb x-rays.」

「  假底有无回声的涂层，以吸收x射线。」

「 An anechoic chamber is a room that absorbs sound.」

「  无回声的房间是一种能吸收声音的房间。」

「 It's something called an anechoic chamber.」

「  这是一种叫做无回声的室的东西。」
