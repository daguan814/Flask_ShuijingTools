今天给大家撸下pass系列

平日里经常听到这样的表达

直接 pass

就是 直接过

所以 pass 就是 通过


然后给大家光速秒杀下系列词


pass

英 /pɑːs/  美 /pæs/

本身即单词

意思就是 通过



bypass

英 /ˈbaɪpɑːs/  美 /ˈbaɪpæs/

by 旁边

pass 通过

从旁边通过

就是 绕过 避开

作名词就是 旁路 支路 的意思



passer

英 /ˈpɑːsə(r)/  美 /'pæsər/

pass 通过

-er 是名词后缀 表示 xx的人

通过的人

也就是 过路人 旅客

也可以指 考试及格的人

平时经常说 通过xx考试

就是 xx考试成绩及格 的意思



repass

英 /riː'pɑːs/  美 /,ri'pæs/

re 再

pass 通过

所以是 再通过



passage

英 /ˈpæsɪdʒ/  美 /ˈpæsɪdʒ/

pass 通过

-age 是名词后缀 表示 xx场所地点

通过的地方

就是 通道 过道



overpass

英 /ˈəʊvəpɑːs/  美 /ˈoʊvərpæs/

over 在xx上面

pass 通过

从上面通过的通道

就是 立交桥 天桥



passable

英 /ˈpɑːsəbl/  美 /ˈpæsəbl/

pass 通过

-able 是形容词后缀 表示 可xx的

所以是 可通过的



passably

英 /ˈpɑːsəbli/  美 /ˈpæsəbli/

pass 通过

-ably 是副词后缀 表示 可xx地

本意是可通过地

衍生出 尚可地 也还过得去地 的意思



passless

英 /'pɑːslɪs/  美 /'pæslɪs/

pass 通过

-less 是形容词后缀 表示 不xx的

不通过的

就是 走不通的 无路可走的



passmark

英 /ˈpɑːsmɑːk/  美 /ˈpɑːsmɑːk/

pass 通过

mark 分数

通过的那个分数

就是 及格分数



passport

英 /ˈpɑːspɔːt/  美 /ˈpæspɔːrt/

pass 通过

port 口岸

能让你通过口岸的东西

就是 护照



password

英 /ˈpɑːswɜːd/  美 /ˈpæswɜːrd/

pass 通过

word 字 词

可以用来通过的字词

就是 密码 口令
