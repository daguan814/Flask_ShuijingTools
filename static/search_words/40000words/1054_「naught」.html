今天给大家撸下naught系列

naugh 是 no 的变体

naugh 发音 = no

作为词根的意思也是 no

也就是 无 没有 零 的意思

词根结尾 + 万能t 不改变意思

所以 naught 依然是 无 没有 零 的意思


然后给大家光速秒杀下系列词


naught

英 /nɔːt/  美 /nɔːt/

本身即单词

就是 无 零 的意思



come to naught

美 /kʌm tu nɔːt/  美 /kʌm tu nɔːt/

come 到达

naught 无

到达无

最终一无所有

也就是 失败 破灭 化为泡影



all for naught

英 /ɔːl fər nɔːt/  美 /ɔːl fɔːr nɔːt/

all 全部

naught 无 零

全部都归于零

就是 徒然 无用
