今天给大家撸下zero系列

Zer 发音 = the

是 the 的变体

变式之前的形式是 the O

那个 O

O 就是 0 的意思

所以 zero 意思就是 0


然后给大家光速秒杀下系列词


zero

英 /ˈzɪərəʊ/  美 /ˈzɪroʊ,ˈziːroʊ/

本身即单词

意思就是 0



zeroth

英 /'ziərəuð, -rəθ, 'zi:-/  美 /'ziərəuð, -rəθ, 'zi:-/

zero 0

-th 是形容词后缀 加在基数词之后 表示 第xx位的

所以是 第零位的



nonzero

英 /nɒn'zɪərəʊ/  美 /nɑːnˈzɪroʊ/

non 表示否定 非

zero 0

所以是 非零的



subzero

英 /sʌb'zɪərəʊ/  美 /ˌsʌbˈzɪroʊ/

sub xx下面

zero 0

所以是 零下

就是天气预报上面经常提到的气温零下



zeroing

英 /'ziərəuiŋ, 'zi:-/  美 /'ziərəuiŋ, 'zi:-/

zero 0

-ing 是名词后缀

所以是 零 归零 名词形式



zeroish

英 /ze'rɔɪʃ/  美 /ze'rɔɪʃ/

zero 0

-ish 是形容词后缀 表示 如xx的

像0的

也就是 接近零的



zeroize

英 /'ziərəuaiz, 'zi:-/  美 /'ziərəuaiz, 'zi:-/

zero 0

-ize 是动词后缀

所以是 化零



zero bit

英 /ˈzɪərəʊbɪt/  美 /ˈzɪroʊbɪt/

zero 0

bit 位

所以是 零位
