今天给大家撸下sec系列

巨常用的高考中考四六级词根

sec 这词根有俩意思

1 切割 2 跟随

一提到 sec

包含 sec 的词各位老铁第一反应肯定都是 second 这个词

小学词汇

sec 本身也是 秒 的意思

second 这词也有俩意思

跟 sec 的俩意思一一对应

1 秒 2 第二的

秒 是什么东西

还不是把一分钟 切 成六十份

其中一份叫 一秒

把时间 切 碎掉 就是一秒一秒的小东西

ond 名词后缀

把时间切碎得到的东西

就是 秒

然后 second 第二个意思

是 第二 的意思

第二名 还不就是 跟随者

跟随在第一名后面的那个人 就是 第二名

sec 跟随 ond 名后表东西

跟随着的东西 就是 第二

所以大家记住 sec 的俩意思了么

一个 切割 一个 跟随

跟 second 的俩意思完全对应


然后之前讲过词根尾辅音的变形规则

c = g = ct

这个很重要

能让你在记词根的时候少花一半的时间

c 跟 g 发音相同

一对清浊音

可以互换

cg 发音相同 只是 c 声带不震动 g 声带震动

所以可以互换不影响意思

然后之前讲过万能 t

任何尾辅音发生变形时都向万能 t 过渡

g 没办法直接加 t

segt

看起来不像英语 太奇怪

所以要先变成 c 再加 t

于是过渡的结果就是 sect

所以 c = g = ct

非常稳定

其中 seq 发音也跟 sec 完全一样

所以这个词根的变式里

sec = seq = seg = sect

证明完毕

所以大家自己记词根书的时候可能觉得很痛苦

我去这么多词根表示一个意思

居然有这么多种变形

其实都有规律

都是一个东西
