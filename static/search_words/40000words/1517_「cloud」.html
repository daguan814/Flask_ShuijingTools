今天给大家撸下cloud系列

cloud 翻译学直译过来发音 = 克劳德

cloud 就是「最终幻想」的主角 克劳德 的名字

「最终幻想」自从 1987 年诞生以来

它的 15 部正统作品 30 多部衍生作品的总销量超过 1.5 亿部

在世纪之交的十几年时间里

「最终幻想」是世界上最成功的的游戏 IP

没有之一

「最终幻想」也是第一款真正影响整个世界的 RPG 游戏

后面风靡全球的诸多 RPG 游戏本质上都是对「最终幻想」创意的抄袭

粉丝们亲切地把「最终幻想」的主角之一 cloud 克劳德 叫做 云片 或者 云片君

给大家随便 po 一段 克劳德 战斗的高光时刻


你可以叫我 克劳德

也可以叫我 cloud

当然还可以叫我 云片君

HUAWEI Cloud 就是 华为克劳德

也就是 华为云

Baidu Cloud 就是 百度克劳德

也就是 百度云

Ali Cloud 就是 阿里克劳德

也就是 阿里云

意思是一样的

完全可以等义替换

所以 cloud 克劳德 就是 云 的意思


然后给大家光速秒杀下系列词


cloud

英 /klaʊd/  美 /klaʊd/

本身即单词

意思就是 云



cloudy

英 /ˈklaʊdi/  美 /ˈklaʊdi/

cloud 云

-y 是形容词后缀 表示 多xx的

所以是 多云的



becloud

英 /bɪˈklaʊd/  美 /bɪˈklaʊd/

be 是动词前缀

cloud 云

乌云蔽日是再常用不过的表达

所以云的动词形式就是 遮蔽 蒙蔽 的意思

「乌云遮蔽了天空」



encloud

英 /ɪnˈklaʊd/  美 /in'klaud/

en 是动词前缀

cloud 云

所以也是 阴云遮蔽 动词形式



cloudage

英 /'klaudidʒ/  美 /'klaudidʒ/

cloud 云

-age 是名词后缀 表示集合名词

所以是 云量



cloudier

英 /'klaudiə/  美 /'klaudiə/

cloudy 多云的

结尾的 y 在后面接成分时要改写为意义相同的 i

-er 是形容词后缀 表示 更xx的

更加多云的

也就是 更加阴暗的



cloudily

英 /'klaudili/  美 /'klaudili/

cloudy 多云的

-ly 是副词后缀 xx地

所以是 多云地 黯然



cloudlet

英 /ˈklaʊdlɪt/  美 /ˈklaʊdlɪt/

cloud 云

-let 是名词后缀 表示 小xx

所以是 小云块 小片云



subcloud

英 /'sʌb'klaʊd/  美 /'sʌb'klaʊd/

sub 下

cloud 云

所以是 云下 云下的



cloudless

英 /ˈklaʊdləs/  美 /ˈklaʊdləs/

cloud 云

-less 是形容词后缀 表示 无xx的

所以是 无云的 晴朗的



unclouded

英 /ʌnˈklaʊdɪd/  美 /ʌnˈklaʊdɪd/

un 表示否定

cloud 云

-ed 是形容词后缀 xx的

所以也是 无云的 晴朗的



cloudward

英 /'klaʊdwəd/  美 /'klaʊdwəd/

cloud 云

-ward 是副词后缀 表示 朝xx地

所以是 朝着云地



overcloud

英 /əʊvə'klaʊd/  美 /ˌoʊvərˈklaʊd/

over 过度

cloud 云

过度的云

就是 布满云层 变阴
