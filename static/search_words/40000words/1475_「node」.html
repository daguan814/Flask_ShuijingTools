今天给大家撸下node系列

node 翻译学直译过来发音 = 搦的 或者 掿的

也就是 捏 握 的意思

任何东西被大力捏握的结果必然是变成一个 小团 小结

所以 node 掿的 就是 结 节 的意思


然后给大家光速秒杀下系列词


node

英 /nəʊd/  美 /noʊd/

本身即单词

意思就是 节 结



nodal

英 /ˈnəʊdl/  美 /ˈnoʊdl/

node 节 结

后面接成分时结尾的e日常脱落

-al 是形容词后缀 xx的

所以是 节的 结的



nodule

英 /ˈnɒdjuːl/  美 /ˈnɑːdʒuːl/

node 节 结

后面接成分时结尾的e日常脱落

-ule 是名词后缀 表示 小xx

所以是 小节 小结



supernode

英 /'suːpənəʊd/  美 /'suːpənəʊd/

super 超

node 节点

所以是 超节点



internode

英 /ˈɪntəˌnəʊd/  美 /ˈɪntərˌnoʊd/

inter 中间

node 节

所以是 节间
